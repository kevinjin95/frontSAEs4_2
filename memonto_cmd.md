# Partie serveur (flask)

- Activer l'environnement virtuel windows
```py
python3 -m pipenv shell
```

- Lancer le serveur:
```py
python3 main.py
```

# Partie client (flutter)

- Supprimer le dossier **build**

- Lancer le front avec:
```py
flutter run
```
